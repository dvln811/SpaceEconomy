T1: Harvestable Resources

All raw materials are extracted from asteroid fields by mining vessels. Every system can have asteroid fields. Within each field are various asteroid types, each yielding a specific ore when mined. Common asteroids appear everywhere; rarer types only spawn in dangerous space.

Asteroid Types

When a miner targets an asteroid, its type determines the ore extracted. Fields in safer space contain only common types. As you venture into lower security, richer and rarer asteroid types appear alongside the common ones.

Field Composition by Security

High-Sec Fields: Veltoite, Corsite, Calite, Carite, Frostine, Silite (6 types)
Med-Sec Fields: All common + Ferranox, Zincara, Stannite, Nitrine, Methrix, Biovex, Nickeline (13 types)
Low-Sec Fields: All above + Titanex, Tungstine, Chromax, Heliox, Xenolith, Sporite, Aminox (20 types)
Null-Sec Fields: All above + Platinox, Aurelite, Palladine, Quartzion, Lithex, Berylox, Kraxolith, Voidstone, Neutrite (29 types)

Every field always contains common asteroids. Richer fields have higher density and more variety.

Volume Design Philosophy

Common = Small Volume (1-2 m3): A small mining ship with 100 m3 cargo can haul 50-100 units of iron ore per trip. This means common ores flow freely through the economy. Stations consuming iron never starve because traders can carry massive amounts cheaply.

Rare = Large Volume (8-20 m3): That same 100 m3 ship can only carry 5-12 units of exotic ore. This naturally limits supply of rare materials, makes them genuinely scarce without artificial mechanics, and creates demand for larger hauler ships on dangerous routes.

Result: The economy self-balances. Common materials are abundant (cheap per unit, easy to move in bulk). Rare materials are genuinely scarce (expensive per unit AND hard to move). You need a big, slow, vulnerable ship to haul meaningful quantities of exotics through null-sec space.

Harvesting Mechanics

Mining Cycle: Target asteroid, activate mining laser, ore extracted per cycle.
Yield: units_per_cycle = mining_power * density_modifier
Cycle Time: 5 seconds base. Reduced by laser tier.
Asteroid Depletion: Asteroids have HP. When depleted, they respawn after a cooldown (common: 30s, rare: 5min, exotic: 30min).
Field Density: 0.5-2.0 multiplier. Higher density = more asteroids per field = faster extraction overall.
